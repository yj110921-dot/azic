# azic

바탕화면에서 새로 시작하는 azic 브랜드 웹페이지 프로젝트입니다.
브랜드 PDF와 향후 콘텐츠 계획 이미지를 바탕으로 구성한 짧은 원페이지입니다.

## 폴더 구조

- `index.html`: 페이지 뼈대와 콘텐츠
- `src/styles`: 디자인 파일
- `src/scripts`: 상호작용 JavaScript
- `assets/images`: 이미지 파일
- `assets/icons`: 아이콘 파일
